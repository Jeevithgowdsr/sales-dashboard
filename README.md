# Sales Dashboard (Next.js 15 + TypeScript + Tailwind + Recharts)

## Project Overview
This project is a simple Sales Dashboard following atomic structural principles.  
It visualizes sales for the years **2022, 2023, and 2024** using mock Kaggle-style data.

## Features
- Atomic component structure
- Sales charts for each year
- Random mock sales data
- Input field to filter minimum sales
- Toggle between Bar, Line, and Pie charts
- Reusable chart component

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- TailwindCSS
- Recharts (Chart Library)

## How to Run Locally
```bash
npm install
npm run dev
